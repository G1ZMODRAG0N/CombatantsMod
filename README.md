# CombatantsMod (Pre-Release)

An Elden Ring PvP focused mod.

## Description (updated description will come on v1.0)

Combatants is a PvP focused mod directed at improving the overall experience for players while also providing new content within the Elden Ring PvP scene. 

You will see many quality of life features and content not available in vanilla game. As of now this mod is in Pre-Release state. Meaning this version is in no means a representation of the finalized version. Source-code, crediting, features, and various different other content will be available for the v1.0 version on release. 

### Dependencies/Requirements

* A official copy of EldenRing
* Including the DLC
* Currently only supports Windows and will have future support for Linux

### Installation/Run

* Please download the latest version from: [https://github.com/G1ZMODRAG0N/CombatantsMod/releases](https://github.com/G1ZMODRAG0N/CombatantsMod/releases)
* Simply place the CombatantsLauncher.exe and the accompanying Combatants folder anywhere on your PC. (Preferably in your Game/Elden Ring/ folder)

### Executing program

* Run using the CombatantsLauncher.exe
* After the application pings to the server you may simply click "Launch Combatants Mod"
